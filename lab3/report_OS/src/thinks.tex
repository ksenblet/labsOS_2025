\section{Выводы}

В ходе лабораторной работы была создана программа для обработки текста, которая использует три процесса, взаимодействующих через общую память (memory-mapped files).

{\bfseries Что получилось:}

\begin{itemize}
    \item Научились работать с общей памятью для обмена данными между процессами
    \item Сделали программу, которая работает одинаково хорошо в Windows и Linux
    \item Данные передаются быстро, так как не нужно их копировать несколько раз
    \item Программа надежно работает и правильно закрывает все ресурсы
    \item Общая память оказалась удобнее и быстрее обычных каналов
\end{itemize}
